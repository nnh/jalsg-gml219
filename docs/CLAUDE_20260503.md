# JALSG-GML219 解析プロジェクト

## 研究概要

- **研究名：** 高齢者急性骨髄性白血病(AML)の層別化により化学療法が可能な症例に対して若年成人標準化学療法の近似用量を用いる第II相臨床試験
- **研究番号：** JALSG-GML219
- **デザイン：** 多施設共同非盲検無対照試験
- **対象：** 65歳以上74歳以下の未治療AML（APLを除く）、PS 0〜2
- **目標症例数：** 128例
- **登録期間：** 2019年11月27日〜2023年7月24日
- **計画書バージョン：** PRT ver2.4（2024年11月20日）
- **解析計画書：** SAP v1（2025年11月30日、作成：国立病院機構名古屋医療センター 齋藤明子）

## 治療スキーマ

| フェーズ | レジメン | 用量 |
|---------|---------|------|
| 寛解導入 | DNR + Ara-C | DNR 45mg/m²(65-69歳)/35mg/m²(70-74歳) day1-3、Ara-C 100mg/m² day1-7（最大2コース） |
| 地固め1 | MIT + Ara-C | MIT 7mg/m²(65-69歳)/5mg/m²(70-74歳) day1-3、Ara-C 200mg/m² day1-5 |
| 地固め2 | DNR + Ara-C | DNR 40mg/m²(65-69歳)/30mg/m²(70-74歳) day1-2、Ara-C 200mg/m² day1-5 |
| 地固め3 | ACR + Ara-C | ACR 16mg/m²(65-69歳)/14mg/m²(70-74歳) day1-5、Ara-C 200mg/m² day1-5 |
| TIT | MTX+Ara-C+PSL | 地固め3終了後、血小板10万/µl以上回復後 |

## 主要・副次評価項目

- **主要：** 全適格例における2年Event-free Survival (EFS)
- **副次：** 5年EFS、FAB/WHO別EFS、2年・5年OS、FAB/WHO別OS、2年・5年RFS（CR例）、FAB/WHO別RFS、CR率、各分子病型別RFS/OS/EFS/CR率、細胞遺伝学的反応性（WT-1）、有害事象、治療実施割合

## 解析対象集団

| 集団 | 定義 |
|------|------|
| FAS | 全登録例から重大なプロトコル違反・未投与例を除いたもの（主解析） |
| PPS | FASから併用禁止薬違反例を除いたもの（参考解析） |
| SAF | FASと同一 |

## エンドポイント定義

| エンドポイント | 起算日 | イベント |
|--------------|-------|---------|
| EFS | 登録日 | 再発・治療不応・あらゆる死亡 |
| OS | 登録日 | あらゆる死亡 |
| RFS | LFS到達日 | 再発・あらゆる死亡 |
